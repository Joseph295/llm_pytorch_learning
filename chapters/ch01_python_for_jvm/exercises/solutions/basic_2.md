# 基础 2 参考答案：广播 shape 心算

规则回顾：**从右往左**逐维对齐；每维上"相等 / 是 1 / 不存在"皆可，否则报错。

**a) `(5, 1, 4) * (3, 4)` → `(5, 3, 4)`**
右对齐：`4 vs 4` ✓相等；`1 vs 3` ✓有 1，扩成 3；`5 vs 不存在` ✓当作 1，扩成 5。

**b) `(2, 3) + (3, 2)` → 报错**
右对齐：`3 vs 2`——不相等、都不是 1，第一维就死刑，不用再看。

**c) `(8, 1) + (1, 8)` → `(8, 8)`**
右对齐：`1 vs 8` ✓扩成 8；`8 vs 1` ✓扩成 8。**双向广播**——两个都被扩展。这个形态在注意力里常见：列向量 op 行向量 = 外积式配对矩阵。

**d) `(6, 7) + (7,)` → `(6, 7)`**
右对齐：`7 vs 7` ✓；`6 vs 不存在` ✓。"给每行加同一个偏置向量"的标准形态（bias 加法就是它）。

**e) `(4, 3, 2) + (3, 1)` → `(4, 3, 2)`**
右对齐：`2 vs 1` ✓扩成 2；`3 vs 3` ✓；`4 vs 不存在` ✓。

验证代码（一行一个）：

```python
import torch
for sa, sb in [((5,1,4),(3,4)), ((2,3),(3,2)), ((8,1),(1,8)), ((6,7),(7,)), ((4,3,2),(3,1))]:
    try:
        print(sa, "+", sb, "->", tuple((torch.zeros(sa) + torch.zeros(sb)).shape))
    except RuntimeError:
        print(sa, "+", sb, "-> 报错")
```

**心法**：拿不准时在纸上把两个 shape 右对齐写成两行，逐列判断。三秒内心算不出来的组合，就该在代码里写 shape 注释了。
